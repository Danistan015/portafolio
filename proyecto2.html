Gestión de Reservas y Clientes

Descripción del Proyecto

Este proyecto es una API RESTful desarrollada con FastAPI que gestiona la relación entre Clientes y Reservas, utilizando MySQL como base de datos y desplegado en Docker Desktop. La arquitectura sigue un modelo en capas, con rutas para operaciones CRUD (Crear, Leer, Actualizar, Eliminar), lógica de negocio separada en servicios, y modelos ORM con SQLAlchemy. Además, se utiliza Alembic para gestionar migraciones de base de datos. La API incluye validaciones como unicidad de datos, limitación de reservas por cliente y control de la relación entre clientes y sus reservas.

Tecnologías Utilizadas

  • FastAPI
  • MySQL
  • Docker Desktop
  • SQLAlchemy
  • Alembic

Desafíos y Soluciones

Durante el desarrollo de este proyecto, uno de los mayores desafíos fue implementar una arquitectura en capas efectiva para manejar las operaciones CRUD entre clientes y reservas. Para resolverlo, se diseñaron cuidadosamente las rutas y la lógica de negocio en servicios separados, lo que permitió una mayor modularidad y mantenimiento del código. Además, garantizar la integridad referencial en la base de datos de MySQL requirió la implementación precisa de relaciones y validaciones en los modelos ORM de SQLAlchemy. Otro reto fue optimizar el rendimiento de las consultas y asegurarse de que las migraciones de base de datos con Alembic se realizaran sin problemas. Estos desafíos se superaron mediante pruebas exhaustivas y ajustes continuos en la configuración y el diseño del sistema.

Conclusión

Este proyecto me brindó una valiosa experiencia en el desarrollo de APIs RESTful con FastAPI, el manejo de bases de datos MySQL y el despliegue en un entorno Dockerizado. Aprendí a implementar una arquitectura en capas para un diseño más limpio y modular, a utilizar SQLAlchemy y Alembic para una gestión eficaz de datos, y a optimizar el rendimiento y la integridad de la aplicación. En general, este proyecto reforzó mis habilidades en desarrollo backend y en la construcción de soluciones escalables y mantenibles.
