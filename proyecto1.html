Gestión Bibliotecaria

Descripción del Proyecto

Este proyecto es una aplicación para la gestión de préstamos, clientes y libros, desarrollada en Java. Utiliza MySQL para gestionar la base de datos con relaciones foráneas que aseguran la integridad de los datos. La aplicación maneja operaciones completas de CRUD (Crear, Leer, Actualizar, Eliminar) y permite la generación de PDFs para llevar un registro detallado de préstamos, devoluciones y el historial de cada cliente. Ofrece funcionalidades de búsqueda avanzada y filtrado para facilitar la localización de libros y clientes. Además, implementa un límite de préstamos por cliente para gestionar mejor la disponibilidad de los libros y prevenir abusos.

Tecnologías Utilizadas

  • Java
  • MySQL
  • PDF Generation (iText)

Desafíos y Soluciones

Durante el desarrollo del proyecto, uno de los mayores desafíos fue manejar la integridad referencial de los datos al gestionar relaciones entre clientes, libros y préstamos en MySQL. Se resolvió implementando claves foráneas y asegurando que las eliminaciones y actualizaciones estuvieran correctamente propagadas. Además, la generación de reportes en PDF representó un reto, que se superó utilizando la librería iText, facilitando la creación dinámica de reportes con los detalles de los préstamos y devoluciones.

Conclusión

Con este proyecto, aprendí a integrar MySQL con Java y aplicar buenas prácticas en el manejo de bases de datos, generación de reportes automatizados y diseño de aplicaciones de gestión. Este proyecto fortaleció mis habilidades en el desarrollo de soluciones completas y robustas para la administración de datos.
